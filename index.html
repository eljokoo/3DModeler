Instructions:

  1. Shape Creation and Deletion:
    • Push 'C': Create a new cube
    • Push 'S': Create a new sphere
    • Push 'T': Create a new torus
    • Push 'H': Create a new tetrahedron
    • Push 'O': Create a new octahedron
    • Push 'I': Create a new icosahedron
    • Push 'D': To delete the currently selected shape
  2. Shape movement
    • Click a shape: To select it in move mode
  3. Shape Scaling:
    • Push 'Q': To increase the size of the selected shape
    • Push 'A': To decrease the size of the selected shape
  4. In Camera Mode:
    • Use arrow keys: To adjust the angle of the camera from its current position
